es6对象新增方法

发表于 2019-10-07

Object.is()

Object.is()可以用来判断两个值是否相等，与 ES5 的区别在于 ES5 中+0 与-0 相等，NAN 与 NAN 不等，而 Object.is()纠正了这个错误。

Object.assign()

Object.assign()用于对象可枚举属性的合并，第一个参数是目标对象，其他参数都是源对象，出现相同属性时后面的对象会覆盖前面的对象。

如果参数不是对象，该方法会将其转为为对象，然后返回，但是原来的值没有变化。

如果无法转化为对象，则报错。

如果非对象参数出现在源对象的位置（即非首参数），那么处理规则有所不同。首先，这些参数都会转成对象，如果无法转成对象，就会跳过。这意味着，如果 undefined 和 null 不在首参数，就不会报错。
其他类型的值（即数值、字符串和布尔值）不在首参数，也不会报错。但是，除了字符串会以数组形式，拷贝入目标对象，其他值都不会产生效果。

以 Symbol 值为属性的对象也会被拷贝到新的对象中。

Object.getOwnPropertyDescriptors(obj)

获取对象自身所有属性的描述（不包括 Symbol），返回一个对象。

由于 Object.assign()在复制访问器属性时是先求值再复制的，因此无法将访问方法也复制过来。

通过 Object.getOwnPropertyDescriptors()和 Object.defineProperties()可以实现复制对象的 set 方法。

Object.getOwnPropertyDescriptors()方法的另一个用处，是配合 Object.create()方法，将对象属性克隆到一个新对象。这属于浅拷贝。

对象原型相关

Object.setPrototypeOf(obj,proto)
如果第一个参数不是对象会被自动转化为对象，但是由于 null 和 undefined 无法转化为对象，因此会报错。

Object.getPrototypeOf()
获取对象的原型，如果参数不是对象会被自动转化为对象

Object.values()

Object.values 方法返回一个数组，成员是参数对象自身的（不含继承的）所有可遍历（enumerable）属性的键值。

如果参数是字符串，字符串会先转化为一个类似数组的对象

如果参数不是对象，Object.values 会先将其转为对象。由于数值和布尔值的包装对象，都不会为实例添加非继承的属性。所以，Object.values 会返回空数组。

Object.entries()
遍历对象，返回键值对数组组成的数组，行为与 Object.values()一直。

将对象转化为 Map 结构

Object.fromEntries()

将一个键值对数组转化为对象，特别适合将 Map 结构转化为对象
