flex布局

发表于 2019-06-02

属性

父元素属性

  • flex-direction 规定主轴方向，子元素的流动方向
    • column 规定y轴为主轴，子元素自上而下流动
    • row 规定x轴为主轴，子元素从左到右流
    • column-reverse 规定y轴为主轴，子元素自下而上流动
    • row-reverse 规定x轴为主轴，子元素从右到左流动
  • flex-wrap 规定子元素换不换行
    • wrap 换行
    • nowrap 不换行,元素按比例缩小
    • wrap-reverse 向上换行
  • flex-flow flex-direction和flex-wrap的简写
  • justify-content 规定主轴上元素的排列方式
    • center
    • flex-end
    • flex-start
    • space-around
    • space-between
  • align-items 规定子元素的对齐方式
    • center 居中
    • flex-end 下对齐
    • flex-start 上对齐
    • strech 拉伸

      子元素属性

  • flex-grow 拉伸倍数
  • flex-shrink 缩小倍数
  • flex-basic 拉伸或缩小的基值,会覆盖width的值
    • 设置为0，宽度也是元素内容，input有最小宽度值，在火狐下141px，不管有没有内容
    • 设置为auto时，即为元素的width值
    • 设置为百分比时会根据父元素的宽度计算宽度
  • flex 上述三个值的按顺序简写
    • auto=> 1 1 auto
    • 1=>1 1 0%
    • 10px=>1 1 10px
    • 40% =>1 1 40%
    • 1 10px =>1 1 10px
    • none => 0 0 auto
    • 默认=>0 1 auto
  • 缩放计算 所有子元素的flex-basic相加,剩余部分按flex-grow或者flex-shrink分配
