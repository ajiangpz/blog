原型

发表于 2019-09-28

object.create会创建一个对象，并把这个对象的[prototype]和指定的对象关联。

属性设置和屏蔽
给一个对象设置属性时，当一个对象的原型链上已经存在该属性，此时会发生三种情况

  • 原型链上的属性是可写的(writable:true)，那么 js 会在对象上创建属性。

  • 原型链上的属性是只读的，那么 js 不会在对象上创建属性，如果在严格模式下，还会报错。

  • 原型链上的属性存在 setter，那么 js 会调用这个 setter，不会将在对象上创建属性。
